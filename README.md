### Quick Start

Clone the repo

```
git clone []
```

Create `.env` in the root of project

```
touch .env
```

Create `VITE_API_BASE_URL` and populate it from `.env.example`

```
VITE_API_BASE_URL = https://www.themealdb.com/api/json/v1/1/
```

Install packages

```
yarn
```

Run project (at http://localhost:3000)

```
yarn dev
```
